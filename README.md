# Dotfiles
My personal config files for (mostly) everything I use.

## Install

```bash
git clone https://github.com/jackcogdill/dotfiles ~/.dotfiles && ~/.dotfiles/install.sh
```

## Contents
- [Bash](home/.bashrc)
- [iTerm2](iterm2/com.googlecode.iterm2.plist)
- [(Neo)Vim](vim/init.vim)
- [Zsh](home/.zshrc)
